These packages provide access to the HP JToolkit for NonStop(tm) Servers. These extensions to the Java(tm) 2 Platform, Standard Edition are specific to NonStop(tm) Himalaya systems. JToolkit includes the Pathsend API for Java, the Enscribe API for Java, the Pathway API for Java and ddl2java.
